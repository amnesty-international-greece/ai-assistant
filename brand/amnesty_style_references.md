# Amnesty International — Brand Style Reference

Compiled from brandhub.amnesty.org and amnistia.org.ar font directory.

---

## Color Palette

| Role | Hex | Notes |
|------|-----|-------|
| **Brand Yellow** | `#FFFF00` | Official Amnesty yellow (confirmed from brand palette) |
| Black | `#000000` | Primary text, headers, logo backgrounds |
| White | `#ffffff` | Body backgrounds, reversed text on black |
| Flame palette | yellow → orange → red gradient | For charts/graphics only |
| Neutral greys | R=G=B values | Always equal RGB channels |

**Rules:**
- Fully opaque colours only — no transparency/opacity variations
- On dark backgrounds: yellow highlights key words only (max 1/3 of text)
- On light backgrounds: maintain black text for contrast
- Campaign-specific colours permitted for cases (e.g. Write for Rights)

## Typography

### Latin scripts: Amnesty Trade Gothic

| Purpose | Variant | Style |
|---------|---------|-------|
| **Action headlines** | Bold Condensed (`BdCn20`) | ALL CAPS, max 70 chars / 3 lines |
| **Quotations** | Condensed Oblique (`Cn18Obl`) | Lowercase, curly quotes |
| **Body / narrative** | Regular / Bold (`AmnestyTradeGothic` / `Bd2`) | Lowercase, bold key points |
| **Editorial headlines** | Regular / Light | Mixed case, highlights in Bold |

### Greek scripts: Roboto
- Body: Roboto Regular
- Headings: Roboto Bold
- Fallback: Arial, Helvetica, sans-serif

### Email fallback stack
`font-family: 'Roboto', arial, helvetica, sans-serif;`
(Web fonts unreliable in email clients — always fall back to system fonts)

## Logo

- Full logo = wordmark + candle + enclosing box (box is mandatory)
- Preferred: black on yellow background
- Acts as a "signature" — keep smaller than half the main message
- Minimum gap from edges: height of candle flame
- Candle can be used standalone if "seamlessly integrated from the outset"

**Don'ts:** Don't change colors/proportions, don't remove the box, don't rotate, don't outline

---

## Design Signature: The 11.25-degree Angle

The candle's bottom cut creates this angle. Used consistently for:
- Framing elements
- Transition/separator lines
- Section dividers
- Icon corner angles

---

## Iconography

- Two-line system (1:3 thickness ratio)
- Round corner radius = thinner line thickness
- 11.25-degree step angles
- Designed to work with text; simplicity prioritized
- Categories: Activism, Climate, People, Environment, Healthcare, Justice, Research, Technology

---

## Spacing

- Logo clear space: candle flame height
- Content max-width: 1468px
- Spacing scale: 20–80 units

---

## Inclusive Design

- Centre marginalized voices — minimal branding when highlighting affected communities
- Avoid stereotypes
- Equity, diversity, amplification as core design principles
