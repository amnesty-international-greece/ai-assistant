# Brand reference - Amnesty International Greece - AI Assistant

For the presentation designer. Match these tokens so the deck looks native to
the platform it describes.

## Palette

| Token | Hex | Use |
|---|---|---|
| Yellow (primary) | `#FFFF00` | Accents, titlebars, highlights, the signature Amnesty yellow |
| Black | `#000000` / `#0a0a0a` | Backgrounds, headers, primary text |
| Paper | `#f5f3ee` | Light backgrounds (warm off-white, NOT pure white) |
| White | `#FFFFFF` | Cards on paper |
| Flame | `#E63B11` | Alerts / errors only - use sparingly |
| Flame gradient | `#FFE600 → #FFA000 → #E63B11` | Optional energetic accent (the candle flame) |

**Signature move:** black + yellow with a hard 4px black rule under yellow bars.
High contrast, confident, activist. Never pastel.

## Typography

- **Titles / headlines:** Roboto Condensed (900 / black weight), tight letter-spacing
- **Body:** Roboto (400/500/700)
- **Mono (codes, protocol numbers):** Roboto Mono

## The mark

The Amnesty **candle wrapped in barbed wire** is the core symbol. Use the
official logo - do not redraw. Place on black, bottom corners, small.

## Tone

Serious, humane, institutional-but-warm. This is a human-rights organisation's
board tooling - competent and trustworthy, never gimmicky.
